Описание работы:
В скачанные файлы (auto.py, config.py, input.txt) должны лежать в одной папке. 
1. В Файле config.py в переменную NAME нужно записать свою фамилию. Она будет использоваться при создании файлов.
2. В input.txt нужно поместить нужные задания. Они должны начинаться с первой строки. Должны быть указаны номера заданий через ")" (как в исходном word файле с заданиями).
3. Сохраните все изменения.
4. Запустите auto.py. Сгенерируется каталог. В нем уже можно писать код.
5. После написания кода можно открыть runner.py.
6. В переменной s указать номер первого и последнего задания.
7. Запустить.
8. В терминале будет выведен готовый формат, который можно будет скопировать в текстовый файл с ответами.

По вопросам тоже писать в тг указанный выше.
